html

html언어는 <태그명> 태그로 구성되어있다 html태그는 대부분의 태그가 시작태그와 마침태그로 구성되어 있다

브라우저는 상단부터 아래로 한줄씩 해석하기 때문에 먼저 해석된 태그는 형제괸계일때 형(누나) 태그가 된다

html파일을 해석하는 주체는 브라우저이며, 브라우저는 다운받은 html파일의 코드 상단부터 아래로 해석하기 시작한다 그리고 해석된 결과를 화면에 보여줌으로써 사용자들의 컨텐츠를 확인할수 있다

태그의 속성

태그의 컨텐츠가 아닌 보충적인 내용을 정의할 수있는 방법

시작태그에서한칸 띄어서 속성명="속성값" 으로 표현한다

속성값을 표현하기 위해서 관례적으로 쌍따옴표 혹은 홀따옴표를 사용한다

속성을 입력하는 순서는 상관이 없다 / 태그의 속성중에는 글로벌속성과 로컬속성이있다

로컬-a태그의href / 글로벌-style="color"

바디에들어가는 태그

제목태그(h1-h6) 설명태그(p) 줄을넘김태그(br) 칸을나눠줄을긋는태그(hr) 주소를적을때는(address)

a태그(링크를걸수있다) 강조태그(글씨체를 강조 (i,em,strong,b) img태크(이미지 파일을 연결)

video태그(영상을 입력할 수 있다) 리스트태그ol,ul (구조와 형태가 비슷한 컨텐츠를 표현)

dl태그(정의형 항목리스트) 효과없는강조태그(span) article(독립적인 컨텐츠)

blockquote(다른사람의것을 인용하여사용할때)

a태그

사용자를 다른페이로 이동시키고 싶을 때 사용

사용자가 a태그부분을 클릭하면 다른페이지로 페이지 이동이 된다

a태그에는 절대경로(어느위치에서나 변하지 않는경로),상대경로가 있다

네이버 절대경로 상대경로 바로가기

img태그

이미지 파일을 연결하여 홈페이지에 보여주는 역할을 하는 태그(img src="" alt="")

src 속성에는 이미지파일까지의 상대경로를 입력

alt 속성에는 이미지를 설명할수있는 대체 텍스트를 입력

img태그 바로가기

video태그

비디오태그는 자식태그로 source src태그를 사용하여 여러형태의 영상을 입력할 수 있다

source src태그로 입력된 영상은 브라우저의 호환성에 의해 여러개중 한개만 재생된다

video
source src="./video/video.mp4" type="video/mp4"
source src="./video/video.webm" type="video/webm"
/video

video태그 바로가기

리스트태그

구조와 형태가 비슷한 컨텐츠를 리스트태그로 표현

리스트사이의 순서의 흐름이 있으면 ol태그를 부모태그로 하고 없으면 ul태그를 부모태그로 한다

li태그는 단독적으로 사용할 수 없으며 반드시 부모태그로 ul/ol태그를 취해야 한다

리스트태그 바로가기

dl태그

어떤 연관된 대상들을 제시하고 설명할 수 있는 맥락을 갖고있는 태그

dl태그의 자식태그로는 dt태그와 dd태그만 사용할 수 있다

dd태그는 반드시 dt태그의 바로다음에 위치해야 한다

dl태그 바로가기
