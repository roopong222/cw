포토샵은 작업 전 레이어창에서 레이어가 잘 선택이 되아있는지 확인할 것

포토샵은 작업하는 내내 레이어창을 주시하며 어떤 레이어가 선택되었는지 확인해야 한다.

포토샵에서 선택영역을 지정한다는 것은 일러스트레이터처럼 특정 오브젝트를 선택하는 개념이 아니라 픽셀영역을 지정한다는 뜻이다

포토샵은 레이어보다 언제나 선택영역이 우선순위가높다 어떤작업이든 선택영역이 지정되어있다면 선택영역안에서 작업을 우선처리하고 선택영역이 지정되어있지 않다면 레이어위주로 작업을 처리한다

선택영역은 레이어를 바꾸더라도 그대로 유지가 되기 때문에 잡업이 끝난 뒤에는 선택영역 해제를 해주는 것을 잊지 말자(ctrl + D)

글자사이 여백 (자간) 평균(-25 ~ -50)

문단의 여백 (행간) 평균(글자크기 기준으로 +,-10)

글자크기가 작을수록 세련된느낌?을줄수있다

출력을할때 쓰기좋은 가장작은크기 7pt

ctrl+T

프리트랜스폼 기능을 사용하여 선택한 레이어 또는 선택한 레이어안의 선택영역을 대상으로 자유롭게 크기조절 및 회전을 할 수 있다

이때 shift키를 누르면 고정되어 있는 가로세로 비율을 깨고 변형할 수 있으며, ctrl키를 누른상태에서 프리트랜스폼의 변형을 할 수 있는 포인트를 클릭 드래그하여 개별적으로도 변형이 가능하다

canvas사이즈 조정하기 [image -> canvas size]

absolute방식과 relative방식

absolute방식은 기복방식으로 현재캔바스의 정확한 크기를 수치로 나타낸다 일러스트레이터와 동일하게 계산식을 사용하여 수치를 조정할 수 있다

relative방식은 상대값을 사용하여 증가하거나 감소할 증감값을 입력하여 기본 캔바스의 사이즈 대비 변경 값만 적용하여 수치를 조정할 수 있다

clone stamp도구

clone stamp 도구는 픽셀을 복사해서 붙여넣기 하는 도구

alt+마우스 좌클릭하여 클릭한 지점을 기준으로 픽셀데이터를 복사한다

복사한 픽셀정보를 필요한곳에 드래그 혹은 클릭하여 사용한다
(이 때,자연스러운 효과를 위해 상단의 옵션창에서 브러쉬의 모양은 soft-round브러쉬를사용한다)

블랜딩

레이어창에서 opacity 값을 조절하여 투명도를 변경해줄 수 있다

opacity 투명도는 언제나 0%-> 완전히 투명해서 안보임 100%->완전히 불투명함

레이어창에서 블랜드방식을 변경해 줄 수 있다 기본값(normal) 그 외로 블랜드방식을 여러방식으로 변경해 굴수있는데

블랜드 옵션이 적용된 레이어는 아래있는 레이어와 혼합이 이루어진다

Image Adjustments Levels

메뉴를 통해서 선택한 레이어의 명도를 조절할 수 있는 level창을 열 수 있다

이 곳에서는 선택한 레이어의 레벨분포를 보여주고 슬라이드를 통해 어두운색상계통(shadow tone) 중간색상계통(mad tone) 밝은색상 계통(highlught tone)의 입력값과 출력값을 조절하여 사진의 필터를 만들어 줄수 있다
