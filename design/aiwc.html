패턴

패턴은 사용하기전에 등록을 먼저 해야한다. 등록하는 방법은 swatches 창을 열어놓고 패턴으로 사용할 object를 선택하여 클릭 드래그로 swatches창에 던지면 등록이 된다.

이미 등록한 패턴을 수정하는 방법

swatches창의 등록되어 있는 패턴 아이콘을더블클릭하면 등록한 패턴을 설정할 수 있는 pattern options 창을 열어볼 수 있다. 이곳에서 크기와 형태 및 모양을 설정할 수 있다.

색상

RGB - red/green/blue/ lught 의 합성치로 색을 표현하는 방법 각각의 불빛의 0~255까지의 수치로 구성되어 있으며 혼합할 수록 밝아지는 가산혼합형태이다

CMYK - cyan/magenta/yellow/black의 잉크 합성치로 색을 표현하는 방법 0~100까지의 수치로 구성되어 있으며 혼합할수록 어두워지는 감산 혼합형태

hsb - 인간을 위한 색을 표현하는 방식 색상/채도/명도로 색을 표현하는 방식으로 쉽게 접근이 가능하다

브러쉬로 등록할 오브젝트를 만든 후 브러쉬 창에 클릭 드래그로 던진다 (이때 가로로 눕혀서 던져야함)

심볼로 등록할 오브젝트를 만든 후 심볼창에 클릭 드래그로 던진다 등록한 심볼은 여러개의 심볼전용툴로 조절해 사용

효과

그림자

Effect-> stylize -> Drop Shadow

선택한 오브젝트를 기준으로 x/y좌표값을 통해 그림자를이동시켜줄수있으며 만드어질 그림자의 투명도를 지정해줄수있다 여기서 blur값은 만들어질 그림자의 뭉개짐 정도를 의미하며 수치가 커질수록 더 넓게 뭉개진다

Effect-> stylize -> Immer Glow

선택한 오브젝트의 안쪽으로 선택색상의 광선을 표현해주는 효과이며 오브젝트 안쪽으로 광선이 나아가는 방식이고 오브젝트 가장자리에서 중앙으로 그리고 광선의 뭉개짐(확산정도)표현하는 blur값 그리고 투명도값을 지정해 줄 수 있다

Effect-> stylize -> Outer Glow

선택한 오브젝트의 바깥으로 선택색상의 광선을 표현해주는 효과이며 오브젝트 바깥쪽으로 광선이 나아가는 방법이다 바깥쪽으로 광선이 표현되기 떄문에 나누어지는 방식은 따로 없으며 blur/opacity 값을 지정할수있다

왜곡

Envelope distort (object -> envelope distort -> make with top object)

mesh 오브젝트 혹은 일반 오브젝트와 type(문자) 오브젝트를 선택해서 상위에있는 오브젝트의 모양대로 type오브젝트의 모양을 왜곡시키는 방법 이 때 mesh혹은일반 오브젝트는 반드시 위에 올라와있어야한다

기출시험에서는 type오브젝트를 선택했을 때 상단의 control창에서 이미 만들어져 있는 mesh프리셋(그물망아이콘)을 선택하여 변형하는 방법이 주로 사용됨

Effect -> free-distort 기능을 사용하여 자유롭게 변형

변형된 정보는 오브젝트를 선택했을 때 Appearance 창에서 확인할 수 있다

path 이동 (offset path)

object -> path -> offset-path 기능을 사용하여 선택한 오브젝트의 패스를 같은 거리만큼 이동하여 복사하는 기능 offset -path 기능을사용한 뒤 선택된 오브젝트는 복사된 오브젝트이다
